# Paysmallsmall

Paysmallsmall landing site with the Founder&apos;s Club program, app ecosystem packages, and WhatsApp application flow.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` or `http://localhost:3000/founders-club`.

## Quality checks

```bash
npm run lint
npm run build
npm audit
```

## Laravel Forge deployment

The production site runs as a Next.js Node process behind Forge's Nginx configuration. Use this deployment script after replacing the site-directory placeholder:

```bash
cd /home/forge/YOUR-SITE-DIRECTORY
git pull origin main
npm ci
npm run check
```

Configure the Forge daemon with this command and restart it after each deployment:

```bash
npm run start -- -p 3000
```

Keep the existing Nginx proxy pointed at `127.0.0.1:3000`. The exact daemon restart command is generated by Forge and should be copied from the site's Daemons panel rather than guessed.
